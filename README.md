# split-tilemap

Small tool to split tilemaps into the individual images

## Get dependencies

`yarn` or `npm install`

## Help

`node index.js`

## Run

Minimal: `node index.js spider.png`

Specifying sizes: `node index.js spider.png -w=64 -h=64`

Output folder: `node index.js spider.png -w=64 -h=64 -o=/MyHome`
